// ABOUT

I am Sándor. Originally from Hungary, currently based on the French Riviera. I enjoy experimenting across different fields: I photograph using legacy digital cameras, develop games on Linux, and build applications and games for Apple platforms (macOS, iOS, iPadOS) using SwiftUI. I am also deeply interested in the connection between artificial intelligence and digital education.

BITO GAMES is my indie game studio. The photos on this site were taken along the Riviera. My developments focusing on the Apple ecosystem will be detailed on a separate page.

// INTERNAL SYSTEMS & AUTOMATION

Workflow-supporting systems and data analysis tools developed for personal use. Focus: Python, desktop Linux environments, and automation. In the age of AI, structural, systems-level, critical, and creative thinking are essential.

Official Gazette Analyzer

Daily data extraction and content analysis from the Hungarian Official Gazette. Automated execution via systemd timer, utilizing the existing Claude environment.

Tech: Python · systemd · Claude

Life Journal Data Logger

Minimalist graphical interface for daily data collection. Structured logging system for long-term personal data analysis.

Tech: Python · Tkinter

Distributed AI Bridge

Asynchronous data exchange between the Linux workstation and the macOS client. The models process information via a locally synchronized file system.

Tech: Syncthing · Claude · Local network
